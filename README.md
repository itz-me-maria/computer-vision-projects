# Multi-Scale Image Analysis

## Overview

This project explores multi-scale image representation using Gaussian and Laplacian pyramids, along with image reconstruction and quality evaluation.

## Key Concepts

* Gaussian Pyramid (image downsampling)
* Laplacian Pyramid (detail extraction)
* Image Reconstruction
* Noise Modeling
* PSNR-based Quality Evaluation
* Image Denoising

## Results

* Multi-scale representation captures image structure at different resolutions
* PSNR analysis quantifies noise impact
* Gaussian filtering improves image quality while introducing slight blur

## Tools Used

* Python
* OpenCV
* NumPy
* Matplotlib
